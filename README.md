# 英语残酷共学

## 介绍

在快速发展与充满无限可能的 Web3 新世界里，英语正在成为每一位学习者不可或缺的核心技能。受限于纯英语交流环境的缺乏，大部分国内学习者的英语能力往往停留在阅读与写作层面。为针对这一痛点，LXDAO 特别发起了「英语残酷共学」 ，旨在为有志于提升英语能力和用好英语的学习者营造一个开放、积极向上的纯英语学习和交流环境。通过有计划的刻意练习，全面提升英语读写尤其是听说的能力。
